# 📦 CRERP -- ERP 自動化工具

本專案為 **ERP 系統相關的自動化工具集合**，\
主要使用 **Python**
開發，用於協助企業進行資料查詢、庫存分析與流程自動化，提高日常作業效率。

此專案包含 ERP 資料處理腳本，例如庫存需求分析、資料整理與報表產出。

# 📄 程式說明

## expiry_requirement.py

此程式主要用於 **ERP 庫存效期需求分析**。

主要功能：

-   查詢 ERP 系統庫存資料
-   分析產品效期需求
-   計算庫存需求狀況
-   協助庫存管理與決策

基本流程：

ERP資料庫 → 資料查詢 → 效期分析 → 需求計算
