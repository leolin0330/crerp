主要使用 **Python**

#  程式說明

## expiry_requirement.py

此程式主要用於 **ERP 庫存效期需求分析**。

主要功能：

-   查詢 ERP 系統庫存資料
-   分析產品效期需求
-   計算庫存需求狀況
-   協助庫存管理與決策

基本流程：

ERP資料庫 → 資料查詢 → 效期分析 → 需求計算
