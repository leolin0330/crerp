# CRERP 自動化工具

CRERP 是一個以 **Python** 開發的 ERP 自動化工具專案， 主要用於 **ERP
資料查詢、庫存分析、效期需求流程自動化**。

本專案透過 Python 自動化處理 ERP 資料，將自動產生資料並寄送 Email 通知，
以減少人工操作並提高作業效率。

------------------------------------------------------------------------

#  專案特色

-    ERP 系統資料查詢
-    自動寄送 Email 通知
-    Python 自動化腳本
-    ERP 流程自動化
-    資料分析與整理
-    資料產出

------------------------------------------------------------------------

# 🧠 技術亮點

本專案展示以下技術能力：

-   Python 自動化開發
-   ERP 系統資料整合
-   SQL / 資料庫查詢
-   Pandas 資料分析
-   流程自動化
