# 📦 CRERP -- ERP 自動化與資料分析工具

CRERP 是一個以 **Python** 開發的 ERP 自動化工具專案， 主要用於 **ERP
資料查詢、庫存分析、效期需求計算與業務流程自動化**。

本專案的目標是減少企業在 ERP 系統中大量人工查詢與整理資料的時間， 透過
Python 自動化與資料分析技術，提高工作效率與資料處理準確度。

------------------------------------------------------------------------

# 📄 程式說明

## expiry_requirement.py

此程式主要用於 **ERP 庫存效期需求分析**。

主要功能包括：

-   查詢 ERP 庫存資料
-   分析產品效期需求
-   計算庫存需求狀況
-   協助庫存管理與決策
