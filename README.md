# ERP 自動化報表通知系統 (ERP Automation Notification System)

## 專案介紹

本專案為企業內部 ERP 系統的 **自動化資料查詢與通知工具**。\
透過 Python 連接 Oracle 資料庫，執行 SQL 查詢後，自動整理資料並透過
Email 發送報表或通知。


------------------------------------------------------------------------

## 主要功能

###  ERP資料自動查詢

透過 Python 連接 Oracle 資料庫並執行 SQL。

功能： - 自動執行 SQL - 取得查詢結果 - 轉換為 pandas DataFrame

使用套件： - oracledb - pandas

------------------------------------------------------------------------


###  Excel 報表產生與寄送

部分腳本會產生 Excel 報表並附加在 Email 中。

流程：
```
SQL 查詢
     ↓
pandas DataFrame
     ↓
輸出 Excel
     ↓
Email 附件 
```
------------------------------------------------------------------------


## 使用套件

  |技術             | 用途|

  |Python           | 自動化腳本|
